WhatsApp Widget Example

This page demonstrates how to embed the WhatsApp widget on your website.

How to Embed

Copy and paste this code into your website before the closing </body> tag:

<script>
    window._whatsappConfig = {
        buttonName: 'Message Us',
        buttonIconSize: 22,
        brandImageUrl: 'YOUR_LOGO_URL',
        brandName: 'Your Company',
        brandSubtitleText: 'Typically replies within seconds',
        buttonSize: 'large',
        buttonPosition: 'right',
        callToAction: 'Start Chat',
        phoneNumber: 'YOUR_PHONE_NUMBER',
        welcomeMessage: 'Hi there ðŸ‘‹',
        prefillMessage: 'Hi, I want to know more!',
        baseUrl: 'https://cdn.jsdelivr.net/gh/RiccardoTran/closerflow-whatsapp-widget@main'
    };
</script>
<script src="https://cdn.jsdelivr.net/gh/RiccardoTran/closerflow-whatsapp-widget@main/embed.js" async></script>
